Word Guessing App

App zum Wörter raten

This app is a word guessing app where users can click an onscreen keyboard to guess a random phrase.
In the java script file there are several arrays which will be choosen randomly.
The phrase will be splitted in letters and will be put on the gamboard.
When the player is guessing a wrong letter he is loosing a live. See project on git-hub

Dies ist eine Spiele-App in der man ein Wort erraten muss. Der Spieler kann auf dem onscreen Keyboard dabei Buchstaben wählen.
Es gibt dabei mehrere Arrays in der Java Script Datei, die zufällig ausgewählt werden
Das zu erratene Wort wird in seine Buchstaben aufgeteilt und auf die Spielfläche gepack
Wenn der Spieler ein falschen Buchstaben errät, verliert er ein Leben. In Git-Hub ansehen
